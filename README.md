# E-Commerce Analytics using Python and SQL

## Project overview

This is an end-to-end analysis of an e-commerce dataset where I used Python, SQL, and a few visualization techniques to dig into sales, customer behavior, payments, product performance, and how sellers were contributing to overall revenue. I split the work between SQL for querying the raw data and Python for the exploratory analysis, and used both together to answer the business questions I cared about.

## What I was trying to answer

- How do sales trends move over time?
- What does customer purchasing behavior actually look like?
- Which product categories are performing best?
- What payment methods do customers prefer?
- How much are individual sellers contributing?
- Where are customers geographically concentrated?
- Which products are the top performers?
- What insights can actually be turned into business decisions?

## Tools I used

Python, SQL (MySQL), Pandas, NumPy, Matplotlib, and Jupyter Notebook.

## About the dataset

The data covers customers, orders, payments, products, sellers, order items, and geolocation, so it's fairly complete from a business standpoint.

## How I worked through it

I started with the raw dataset and cleaned it up, then built a proper database from it. From there I ran the SQL analysis first to get the numbers right, followed by the Python EDA to visualize and validate what the SQL was showing, and finished by pulling everything together into business insights.

## SQL work

I calculated total orders and total customers, broke down revenue, identified the top-selling categories, looked at how customers were distributed, analyzed payment preferences, checked seller performance, and worked out the average order value.

## Python work

On the Python side I handled data cleaning, checked for missing values, ran a correlation analysis, looked at sales trends over time, did some customer segmentation, evaluated product performance, looked at how payments were distributed, and built out the visualizations to support all of it.

## Visualizations

Screenshots of the following are in the Images folder: sales trend, monthly revenue, top categories, payment methods, a correlation heatmap, and customer distribution.

## What I found

I was able to pinpoint the best-selling product categories, get a clearer picture of how customers actually purchase, see which products contribute the most revenue, understand which payment methods customers lean toward, identify the sellers performing above the rest, and map out where customers are concentrated geographically.
